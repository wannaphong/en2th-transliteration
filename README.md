# English2Thai Transliteration
English to Thai transliteration

> python inference.py

output:

```
English         | Thai (Predicted)
-----------------------------------
hello           | เฮล'โล         
computer        | คัมพิวเตอร์    
bangkok         | แบง'อค         
pie             | พี             
cat             | แคท            
unknow          | อังคโว         
sunset          | ซัน'เซท 
```

**Run index.html**

> python -m http.server 8000

and go to http://127.0.0.1:8000
