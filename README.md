# English2Thai transliteration
English to Thai transliteration

> python inference.py

output:

```
English         | Thai (Predicted)
-----------------------------------
hello           | เฮลโล          
computer        | คอมพิวเตอร์    
bangkok         | แบงก์็อก       
padthai         | แพดเทด         
cat             | แค็ต   
```

**Run index.html**

> python -m http.server 8000

and go to http://127.0.0.1:8000
